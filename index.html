Navegadores web

Un navegador web (en inglés,web browser)es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que esos puedan ser vistos. >

La funcionalidad básica de un navegador web es permitir la visutalización de documentos de sexo, posiblemente con recursos multimedia incrustados

Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponibles para macOS,iPadOA e iOS.

Mozilla Firefox es un navegador web libre y de código abierto desarrolado para Linux, Android,iOS,macOS y Microsofot Windows coordinadco por la Coportadción Mozilla y la Fundación Mozilla.

Google Chrome es un navegador web de código cerrado desarrolado por Google, aunque derivado de proyectos de código abierto
